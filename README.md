-- But du projet  --

Première vCard après une vie a porter des cv papiers ou distribuer une copie virtuel du cv papier.
Le but de cette vCard est de me présenter, tel que je suis
le projet est un devoir en cour de développement d'application web et pour l'intant ne comporte que du html et css car.. j'apprends
La page a été développé pour etre sur un seul frame affiché à l'écran.  un style page de bd lui est présentement appliqué mais peut etre changé dans
stylebackground.css 
les items dans la page sont des boites qui se scroll si elles devraient étirer la page.
